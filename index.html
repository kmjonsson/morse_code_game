About

This is a simple web/"mobile" "game" to learn morse code.

I created this for my own learning. If you find it usefull, use it :-)

Keyboard

Real keyboard can be used instead for pushing the virtual keyboard.

Enter/Space can be used to push the start button.

Esc/Enter/Space can be used to close the score board.

Esc can be used to abort game and leave game.
